# Dark Mode Quick Start Guide

## What is Dark Mode?

Dark Mode is a display setting that uses light-colored text and icons on a dark background, making it easier to read in low-light environments and reducing eye strain during extended reading sessions.

## Tips for the Best Reading Experience

### Combine with Font Size
Dark mode works great with the font size control! Try:
- Dark theme + Large font for comfortable night reading
- Light theme + your preferred size for daytime

### Reduce Eye Strain
If you read for long periods:
1. Use dark mode in low light
2. Take breaks every 20 minutes
3. Adjust font size to comfortable level
4. Keep screen at arm's length

### Battery Saving
On phones with OLED screens (iPhone X and newer, many Android phones):
- Dark mode can significantly extend battery life
- Black pixels on OLED screens use almost no power

## Keyboard Shortcuts

While there are no dedicated keyboard shortcuts yet, you can:
1. Press `Tab` to navigate to the theme buttons
2. Press `Enter` or `Space` to select a theme
3. Use `Shift+Tab` to go back

## Accessibility

The dark mode feature is designed to be accessible:
- **Screen Readers**: All buttons have clear labels
- **Keyboard**: Fully navigable without a mouse
- **Visual**: Clear active state shows selected theme
- **Announcements**: Theme changes are announced to screen readers

## Troubleshooting

### Theme Not Saving?
- Make sure cookies/storage is enabled in your browser
- Try clearing your browser cache
- Check if you're in private/incognito mode (settings won't save)

### Can't See the Buttons?
- The theme controls are in the reader header
- Look between the chapter name and font size controls
- On mobile, they may wrap to a second line

### Auto Mode Not Working?
- Check your device's dark mode settings
- Make sure your browser is up to date
- Try manually selecting light or dark mode instead

## Mobile Experience

On mobile devices:
- Theme buttons are slightly smaller but still easy to tap
- Dark mode is especially useful for nighttime reading
- Auto mode syncs with your phone's dark mode schedule

## Print and Screenshots

When you print a page or take a screenshot:
- Printed pages always use light theme (better for printing)
- Screenshots will show your current theme
- Dark mode won't waste printer ink

## Frequently Asked Questions

**Q: Does dark mode work on all pages?**  
A: Currently, dark mode is available on the reader pages. We may expand it to other pages in the future.

**Q: Can I customize the dark mode colors?**  
A: Not yet, but we've carefully chosen colors for optimal readability and reduced eye strain.

**Q: Will my theme choice work on other devices?**  
A: Theme preferences are saved locally on each device. If you switch devices, you'll need to set your preference again.

**Q: Does dark mode use less battery?**  
A: On devices with OLED screens (many modern phones), yes! Dark pixels use less power.

**Q: Is dark mode better for my eyes?**  
A: Dark mode can reduce eye strain in low-light conditions, but it's a personal preference. Use what's most comfortable for you!
